# HTML

html은 사실 이미 알고있는 내용이다. 다만 복습의 의미로 다시 공부하게 되었고 효율을 위해 빠르게 훓고 넘어가도록 하겠다.

html은 태그로 이루어져있다. strong, h1, h2 , ...

## 의미론적 코드의 중요성성
br - 줄바꿈, p - 단락 -> p태그가 코드를 의미론적으로 표현할수 있기 때문에 더 가치있는 정보로 만들어 줌
검색엔진은 웹 페이지에 담긴 HTML 코드를 분석하고 태그에 근거하여 정리정돈을 함. 따라서 의미에 맞게 HTML을 작성하는 것이 중요하다

img `<img src="">` - 이미지 태그

## 부모 자식 태그
```
<ol>
    <li>HTML</li>
    <li>CSS</li>
    <li>JS</li>
</ol>
<ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JS</li>
</ul>
```

## title
웹 페이지의 제목을 결정
```
<title>WEB</title>
```

## UTF - 8
우리가 작성하는 파일은 UTF-8이란 방식으로 저장되어 있고, 열 때도 UTF-8로 열어야 하기 때문에 이런 코드를 작성함
```
<meta charset="UTF-8">
```

## html, head, body
가장 큰 코드 구조는 html태그 안에 head와 body가 들어있는 구조
